## Game Explanation
In this exergame, the player uses body movements to control the character. The player must run in place to move forward in the game world. Different physical movements are used to interact with obstacles and challenges. For example, the player can squat to dodge obstacles and jump to overcome barriers.

During the game, players can collect coins  to earn points and unlock rewards. The game also includes mini challenges such as squats, jumping jacks, knee raises, and stretches. Completing these challenges allows the player to unlock new worlds and progress further in the game.

This system encourages players to stay physically active while making exercise fun and engaging.
