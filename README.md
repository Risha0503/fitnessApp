## Game Explanation

In this exergame, the player uses body movements to control the character. The idea is that the player runs in place to move forward in the game world, and performs different movements to interact with obstacles and challenges. For example, the player can squat to dodge obstacles and jump to overcome barriers.

During the game, players can collect coins to earn points and progress. The game also includes mini challenges such as squats, jumping jacks, knee raises, and stretches. Completing these challenges allows the player to unlock new worlds and continue in the game.

## Current Status

At the moment, body movement detection is not yet implemented. The current version of the app includes:

- A working timer system  
- Distance tracking (KM count)  
- A challenge screen that initializes during gameplay  

These features simulate the gameplay flow, but the physical movement input will be added in a future version.

## Future Improvements

##  Purpose

This game is designed to encourage players to stay physically active while making exercise fun and engaging.
